# AtomSpectra Gateway — мобильный веб-UI (демо)

Интерактивный прототип мобильного интерфейса для WiFi-шлюза гамма-спектрометра
**KB Radar «Atom Spectra»** на ESP32-S3.
См. основной проект: <https://github.com/VibeEngineering-LLC/atomspectra-waterfall-esp32>

Один самодостаточный файл `index.html` — работает офлайн, без сборки и зависимостей.

## Что внутри
- **Портрет** — полное приложение, 6 вкладок (Спектр, Водопад, Мониторинг, Спектры, Система, Сервис),
  нижняя таб-панель, живые цифры (CPS, время накопления, счётчики).
- **Спектр** — библиотека нуклидов (Cs-137, K-40, Co-60, Am-241, I-131, Na-22, Ra-226, Tl-208)
  с маркерами энергий на графике; полноэкранный режим.
- **Водопад** — Старт/Стоп/Сброс, интервал записи, спектрограмма с оверлеем нуклидов,
  спектр среза; свайп вверх → полный экран с навигацией, перекрестием и зумом.
- **Мониторинг** — горизонтальный режим: жесты (перетаскивание — прокрутка, щипок — зум,
  тап — курсор), окно скользящего среднего в %, шкалы CPS и времени.
- **Горизонтальные режимы** спектра и мониторинга (стопкой справа).

## Публикация через GitHub Pages
1. Загрузите папку `atomspectra-mobile/` в репозиторий.
2. Settings → Pages → Source: ветка + корень (или `/docs`).
3. Страница откроется по адресу `https://<owner>.github.io/<repo>/atomspectra-mobile/`.

Данные в демо — синтетические (реального прибора не требуется).
