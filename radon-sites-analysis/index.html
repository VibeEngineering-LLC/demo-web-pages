Опорные модели

Воронка действия

Check/Test -> Measure -> Act/Protect

UKradon, Take Action - ведёт от вопроса есть ли радон? к измерению и снижению.

Сегментация аудитории

дом / работодатель / профи / регион

UKradon Sector Guidance, Take Action Resources - один контент под разные роли.

Коммерческий слой

продукты + услуги + корзина поверх контента

Radonova, Ecosense, Airthings - монетизация на базе доверия от просветительского ядра.

Митигация-провайдер

услуги + обучение + продукты + онлайн-консультация

American Radon Mitigation, SWAT - подрядчик как контент-эксперт (YouTube/DIY/CE-курсы).

C. Проект RU-сайта: карта разделов

Костяк - воронка Узнать -> Измерить -> Снизить (адаптация Test/Protect/Learn), поверх - УТП контура (онлайн-оценка риска по показаниям прибора) и подключаемые модули Магазин / Услуги. Синяя рамка - контент-разделы, жёлтая - pluggable-модули (старт: заглушки скоро).
1. О радоне - контент-ядро, SEO под Яндекс
WP: Страницы WP + таксономии тем
  • Что такое радон и откуда берётся
  • Чем опасен: рак лёгкого, годовая доза - нормы WHO/EPA/IAEA/ICRP/НРБ-99/EU-BSS
  • Радон в доме / школе / на работе
  • Карта радоноопасности регионов РФ
  • FAQ
2. Измерить радон - воронка -> сервис
WP: Страницы + кастомный шаблон/виджет калькулятора
  • Как измерять - приборы, длительность, где ставить
  • Приборы: RadonEye, AtomFast, Radex - обзор / сравнение
  • ★ Онлайн-оценка риска ★ УТП - загрузка показаний -> годовая доза + риск = УТП контура
  • Интерпретация результатов - Бк/м3 -> действие
3. Снизить радон - Act
WP: Страницы WP
  • Методы снижения - SSD/депрессуризация под плитой, проветривание, HRV, герметизация
  • Когда нужна профессиональная митигация
  • Найти специалиста / заказать услугу - -> модуль Услуги
4. Магазин МОДУЛЬ - pluggable-модуль, включается позже
WP: WooCommerce Products + категории
  • Детекторы радона - континуальные мониторы
  • Тест-киты - пассивные, лаб-анализ
  • Митигация: вентиляторы, комплектующие
  • Корзина / Оформление / Доставка / Оплата
5. Услуги МОДУЛЬ - pluggable-модуль, включается позже
WP: CPT service или Woo-booking
  • Профессиональное измерение - выезд
  • Лабораторный анализ
  • Митигация / проектирование системы
  • Онлайн / виртуальная консультация - расширение сервиса оценки риска
  • Форма заявки / калькулятор стоимости
6. Блог / Новости - SEO-трафик, awareness-кампании
WP: Записи WP + рубрики
7. Ресурсы
WP: Страницы + вложения + встраивание видео
  • Методички, нормы РФ
  • Видео / DIY - по образцу American Radon Mitigation
  • Для врачей / школ / застройщиков
8. О нас / Контакты - доверие
WP: Страницы + форма
  • Методология, приборная база
  • Обратная связь

Платформа: WordPress + WooCommerce

  • Разделы 1-3, 6-8 -> страницы WP + записи (блог); таксономии для сегментов аудитории (дом/работа/профи/регион) и тем.
  • Раздел 4 Магазин -> WooCommerce Products (детекторы / тест-киты / митигация); корзина, чекаут, оплата, доставка - штатные Woo. Старт: каталог скрыт / скоро.
  • Раздел 5 Услуги -> CPT service (или Woo-booking); форма заявки + калькулятор стоимости. Онлайн-оценка риска - кастомный шаблон/плагин-виджет.
  • Расширяемость: Woo и CPT добавляются без переделки темы; меню-пункты /shop, /services зарезервированы с самого начала.
  • SEO под Яндекс: Schema.org (Product, Organization, FAQPage), человекочитаемые ЧПУ-URL; переиспользовать методику seo-audit-ru.
