AtomSpectra Waterfall — экосистема WiFi-мониторинга гамма-спектрометра

Четыре инструмента вокруг гамма-спектрометра KB Radar «Atom Spectra»: ESP32-S3 превращает USB-прибор в WiFi-точку доступа со спектром в браузере, десктопный вьюер разбирает многочасовые водопады, рекордер непрерывно пишет данные на диск, а прошивальщик заливает всё это на плату в три клика.

Последний релиз шлюза Все репозитории семейства Организация VibeEngineering LLC

Как это работает

Плата подключается к спектрометру по USB, дальше данные расходятся по WiFi: в браузер, по TCP-мосту в BecqMoni/AtomSpectra, и через pull-клиент — на диск для долгих записей.

Прибор KB Radar Atom Spectra 8192-канальный спектр, USB (shproto)
→
Шлюз atomspectra-waterfall-esp32 ESP32-S3, USB OTG Host, WiFi AP/STA
→
Приём данных Web UI · TCP:8234 · wf-recorder браузер, BecqMoni/AtomSpectra, pull-клиент на ПК
→
Анализ waterfall-viewer 3D/2D спектрограмма, срезы, библиотека нуклидов

Состав семейства

Каждый проект — самостоятельный релизный цикл со своим тегом версии. Ниже — актуальный релиз на дату публикации; свежие версии смотрите по ссылке репозитория.

atomspectra-waterfall-esp32

прошивка

WiFi-шлюз на ESP32-S3: спектр в браузере в реальном времени, экспорт в BecqMoni / InterSpec / N42.42 / ЛСРМ, TCP-мост для десктопного ПО, запись водопада на flash.

  • Живой спектр в браузере без ПК рядом с прибором
  • Полевой режим (Outdoor AP) + captive-страница
  • Хранит до ~400 спектров на плате
firmware-v1.1.2 репозиторий релизы

waterfall-viewer

Windows · Python

Десктопный анализатор спектрограмм: 3D-поверхность и 2D-карта Время×Энергия, срезы, сечения, прямоугольные выборки, библиотека из 21 нуклида для маркеров энергий.

  • Формат прибора .aswf, а также RadiaCode .rcspg и N42.42
  • Потоковая загрузка — суточные записи без переполнения памяти
  • Готовый .exe, установка Python не требуется
v0.1.9 репозиторий релизы

wf-recorder

Windows · Python

Настольный pull-клиент: непрерывно забирает сегменты водопада со шлюза, сшивает в единый файл с контролем целостности (CRC32 + порядковые номера), логирует температуру.

  • Удаляет сегмент на плате только после fsync на диск ПК
  • Авто-ротация файла при смене формата прошивки
  • Только stdlib — .exe без внешних зависимостей
v0.2.1 репозиторий релизы

atomspectra-waterfall-esp32-flasher

Windows · Python

Универсальный прошивальщик плат ESP32: тянет актуальную прошивку с GitHub Releases и заливает по USB в несколько кликов, без командной строки.

  • Поддерживает 4 проекта: AtomSpectra, AtomFast, Radex, RadonEye шлюзы
  • Автоопределение COM-порта, кэш прошивок локально
  • Автономный .exe (~124 МБ), Python не нужен

Перед использованием

!

Прошивка и рекордер взаимодействуют с прибором напрямую. Шлюз посылает спектрометру штатные команды управления и при включённой опции очистки может стереть накопленный на приборе спектр; сохраните важные данные штатным ПО заранее. Флешер перезаписывает flash-память платы — выбирайте COM-порт внимательно, посторонние USB-UART-устройства прошивать нельзя. Подробности и полный отказ от ответственности — в README каждого репозитория.
