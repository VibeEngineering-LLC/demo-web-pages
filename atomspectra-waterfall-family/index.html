4 репозитория

atomspectra-waterfall-esp32

прошивка · ESP32-S3

WiFi-шлюз между прибором и сетью: подключается к спектрометру по USB, отдаёт спектр (8192 канала) в браузер в реальном времени, экспорт в BecqMoni / InterSpec / N42.42 / ЛСРМ, TCP-мост порт 8234, хранение до ~400 спектров на flash.

firmware-v1.1.211.07.2026 репозиторий релизы

waterfall-viewer

desktop · Windows/Python

Просмотрщик спектрограмм: 3D-поверхность и 2D-карта Время×Энергия, срезы, сечения, выборки ROI, библиотека 21 нуклида для маркеров энергий. Форматы: .aswf, RadiaCode .rcspg, N42.42. Потоковая загрузка суточных записей.

v0.1.909.07.2026 репозиторий релизы

wf-recorder

desktop · Windows/stdlib

Pull-клиент для непрерывной записи: забирает сегменты водопада со шлюза, сшивает в файл .aswf, удаляет на плате только после fsync на диск, проверка целостности CRC32 + порядковые номера, лог температуры.

v0.2.111.07.2026 репозиторий релизы

atomspectra-waterfall-esp32-flasher

desktop · Windows/Python

Прошивальщик плат ESP32: тянет актуальную прошивку с GitHub Releases, заливает по USB, автоопределение COM-порта. Также прошивает шлюзы AtomFast, Radex, RadonEye — 4 проекта в одном инструменте.

Шлюз и флешер взаимодействуют с прибором напрямую. Опция очистки спектра при старте шлюза стирает накопленные на приборе данные — сохраните их штатным ПО заранее. Флешер перезаписывает flash выбранной платы: проверяйте COM-порт, посторонние USB-устройства прошивать нельзя. Полный текст условий — в README каждого репозитория.
