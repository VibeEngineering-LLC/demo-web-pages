Сквозные принципы

01
actor ≠ verifier

Конструктор и скептик — разные агенты. Финальный вердикт по факту, необратимое действие — за оператором.

02
Local-first делегирование

Bulk-разбор, шаблонная генерация и суммаризация уходят на локальную модель; frontier — для решений и синтеза.

03
Границы зон

Контур правит только свою папку. Смежные проекты не трогает даже при сходстве — вместо этого пишет задание владельцу.

04
Тройная избыточность

Навыки и незаменимые артефакты — в облачном бэкапе, локальном git и на GitHub одновременно.
