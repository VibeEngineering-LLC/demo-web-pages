Коротко о методе

  • Что решает. По одному измерению C(t) за время t метод относит помещение к одному из трёх исходов с надёжностью решения ≥ 95 %: соответствует нормативу, не соответствует (нужны защитные мероприятия) или решение неопределённое (измерение продолжить).
  • Главная идея. Основной вклад в неопределённость решения даёт временная неопределённость UV(t) (радон сильно меняется во времени), а не приборная UD. Поэтому длительность измерения важнее точности прибора.
  • Практический вывод статьи. Краткие измерения (4–6 суток) почти не уступают длительным (2–3 месяца) по доле надёжных решений (в среднем ≈ 80 % по 26 странам), но дешевле более чем на порядок.
  • Режимы. Таблица UV(t) ниже — для нормального режима эксплуатации/вентиляции (минимум 4 суток). Для закрытого режима — минимум 2 суток (отдельная база, здесь не приводится).
(a) соответствует:  C(t)·[1 + √(UV² + UD²)] < CRL
(b) не соответствует:  C(t)/MF(t)·[1 − UD] > CRL
(c) иначе — решение неопределённое, измерение продолжить.

Пороги: CAL(1) = CRL/[1+√(UV²+UD²)]  ·  CAL(2) = MF(t)·CRL/(1−UD). Формулы (1),(2),(7),(8) статьи.
